## 📊 Overview

A comprehensive **real-time banking analytics dashboard** built with Microsoft Power BI for YoursBank, providing deep insights into customer behavior, transaction patterns, and financial risk metrics. This dashboard enables data-driven decision-making through interactive visualizations and real-time monitoring.

## ✨ Features

### 📈 Key Performance Indicators
- **Total Amount**: ₹7.65M with -66.29% vs Previous Year and also varies with different years
- **Total Transactions**: 806 with -66.42% vs Previous Year and also varies with different years
- **Avg Transaction Value**: ₹9.49K with 0.37% vs Previous Year and also varies with different years

### 📊 Interactive Visualizations & Tools
- **Dynamic Slicers & Timelines**: Filter data instantly by Year, Month, Customer Segment, and State.
- **PivotCharts**: Interactive charts for monthly fee trends, transaction status, and customer segments.
- **Conditional Formatting**: Automatically highlight high-risk transactions and top-performing segments.
- **Data Validation**: Drop-down lists for controlled metric and category selection.

### 📋 Detailed Transaction Table
Complete audit trail including:
- Transaction ID and Date
- Customer Name and Details
- Transaction Type and Status
- Financial Metrics (Amount, Fees, Tax)

## 🛠️ What I Did in Excel
- **Data Cleaning & Transformation**: Used **Power Query** to import, clean, and transform raw transaction data, removing duplicates and handling missing values.
- **Advanced Formulas**: Implemented complex formulas like `XLOOKUP`, `SUMIFS`, `IFS`, and `INDEX-MATCH` to calculate dynamic KPIs and Year-over-Year (YoY) growth percentages.
- **PivotTables & PivotCharts**: Built multiple PivotTables to aggregate data by customer segment, state, and transaction status, then visualized them using PivotCharts.
- **Interactive Dashboard Design**: Designed a clean, user-friendly dashboard layout using **Slicers** and **Timelines** to allow dynamic filtering by Year, Occupation, and Category.
- **Conditional Formatting**: Applied conditional formatting to data tables and KPI cards to automatically highlight high-value transactions and negative YoY trends.

## 🛠️ What I Did in Power BI
- **Data Modeling**: Designed a robust **Star Schema** data model, creating optimized relationships between the fact table (Transactions) and dimension tables (Customers, Calendar, Dynamic Metrics).
- **DAX Measures**: Wrote advanced **DAX (Data Analysis Expressions)** formulas to calculate total amounts, averages, and complex Time Intelligence functions for Year-over-Year (YoY) variance calculations.
- **Power Query (M Language)**: Utilized Power Query Editor to connect to data sources, clean the data, unpivot columns, and create custom columns before loading it into the model.
- **Interactive Visualizations**: Created a multi-page report featuring dynamic charts, custom tooltips, and drill-through pages for deeper transaction-level analysis.
- **UX/UI Design**: Implemented interactive **Slicers**, bookmarks, and buttons to create a seamless, app-like navigation experience for the end-user.
